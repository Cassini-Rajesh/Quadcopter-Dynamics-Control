## Overview
Simulated quadcopter 6-DOF nonlinear dynamics using Lagrangian 
mechanics in MATLAB and designed PID controllers for pitch, roll, 
yaw, and altitude control in Simulink.

## Tools & Methods
- MATLAB/Simulink
- Lagrangian mechanics
- 6-DOF nonlinear dynamics
- PID control design
- ODE45 numerical integration
- State-space stability analysis

## Key Results
- Implemented and tuned PID controllers for pitch, roll, 
  yaw, and altitude
- Validated stability via state-space analysis and 
  trajectory plots
- Modeled hover, ascent, and roll conditions
